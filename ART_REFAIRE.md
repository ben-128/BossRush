## Post-traitement obligatoire (tous les heros)

Apres generation de chaque hero, faire un **inpainting cible** :
1. **Visage** : regenerer en zoom le visage si trop flou/generique/lisse. Insister sur les imperfections specifiques du perso (cicatrices, asymetrie, expression). Le visage doit etre lisible a taille carte imprimee.
2. **Details armure/equipement** : inpainting sur les zones d'armure, armes, emblemes si textures trop "fondues". Ajouter nettete et bords durs.
3. **Contraste figure/fond** : si le personnage se perd dans le decor, assombrir le fond ou ajouter un accent de couleur froide (reflet bleu acier, lueur magique) pour creer du contraste avec la palette chaude.
4. **Palette** : verifier que la couleur thematique du heros domine bien. Si tout est brun/orange monotone, injecter la couleur thematique plus fort (rouge vif guerrier, bleu electrique mage, etc.) et assombrir le fond pour que les elements de carte (gemmes, bordure) ressortent.

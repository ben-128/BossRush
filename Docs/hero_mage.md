# Mage - Illustration

## Fichier
`BossRush/Assets/Raw/Heros/Mage.png`

## Statut
Fond corrige (gris-beige plat). Densite encore un peu elevee (~60-65%) vs Guerrier (~50%).

## Score qualite : 6.5/10

## Corrections a faire

### 1. Visage (PRIORITE)
Le visage est trop petit, trop symetrique, trop lisse. A taille carte ca sera une tache. Besoin de plus de detail, d'asymetrie, de personnalite.

**Prompt (uploader l'image) :**
```
This is an existing ink engraving illustration. Recreate it as a
PIXEL-PERFECT copy with this specific improvement only:

FACE: Make the face LARGER and much more detailed. Sharp angular
features, hollow cheekbones more pronounced, piercing asymmetric
eyes - the left eye slightly larger with a faint blue glow (arcane
burnout). Thin severe lips with an asymmetric smirk - one corner up,
one corner down. The burn mark on the temple should be clearly
visible. Silver streaks in hair more defined. The face must be the
SHARPEST most detailed area of the entire image. Paint it with rough
thick strokes, NOT smooth. She should look unsettling and intense,
not pretty.

Keep EVERYTHING else identical - same pose, same composition, same
lightning, same staff, same crystal, same arcane circle, same sky,
same mountains. Same flat smooth grey-beige background color. Same
blue accent color.
```

### 2. Cheveux bleus (IMPORTANT)
Les cheveux entierement bleus font "anime/jeu video" — pas raccord avec le style gravure serieux du Guerrier. Le bleu devrait etre plus subtil.

**Prompt (uploader l'image) :**
```
This is an existing ink engraving illustration. Recreate it as a
PIXEL-PERFECT copy with this specific improvement only:

HAIR: The hair should be mostly BLACK ink with only subtle blue
streaks/highlights, not entirely bright blue. The hair should look
like dark hair with a few silver-blue magical streaks woven through
it, wind-whipped and wild. Most of the hair mass should be black
ink hatching, with blue used sparingly as accent only.

Keep EVERYTHING else identical.
```

### 3. Mains/doigts (IMPORTANT)
La main gauche a des doigts visibles — probleme IA classique. Il faut les cacher.

**Prompt (uploader l'image) :**
```
This is an existing ink engraving illustration. Recreate it as a
PIXEL-PERFECT copy with this specific improvement only:

LEFT HAND: The hand channeling lightning should be completely
ENVELOPED in bright crackling blue magical energy. The blue glow
should be so intense that individual fingers are not visible at all
- just a silhouette of the hand shape inside dense blue lightning
plasma. No bare fingers visible.

Keep EVERYTHING else identical.
```

### 4. Densite d'encre (OPTIONNEL)
Encore un cran trop dense par rapport au Guerrier. Les hachures sont plus fines et serrees.

**Prompt (uploader l'image) :**
```
This is an existing ink engraving illustration. Recreate it as a
PIXEL-PERFECT copy with this specific improvement only:

HATCHING: Make the hatching strokes BOLDER and more SPACED OUT,
like woodcut style. Currently the hatching is too fine and dense.
Open up more light areas in the sky and ground to let the background
breathe. Target approximately 50% ink coverage, not 65%. The image
should feel lighter and airier overall.

Keep EVERYTHING else identical.
```

### 5. Eclairs bleus
Les eclairs sont un peu trop "digital clean" par rapport aux hachures du reste. Idealement ils devraient avoir un cote plus "grave" aussi.
